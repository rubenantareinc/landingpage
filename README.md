# Ruben Arena — GitHub Pages Landing Site

A minimal, premium landing page for Ruben Arena with an intro animation, glassmorphism cards, and subtle glow blobs. Built with vanilla HTML/CSS/JS and ready for GitHub Pages.

## Update the CV
1. Replace `assets/Ruben_Arena_CV.pdf` with your latest CV PDF (keep the filename the same).
2. Commit and push.

## Update links
Edit `index.html` and replace the placeholder URLs:
- LinkedIn: `https://www.linkedin.com/in/`
- Email: `mailto:hello@example.com`
- Project PR: update the PR link in the first project card.
- Project repositories: update if needed.

## Enable GitHub Pages
1. Go to **Settings → Pages**.
2. Under **Build and deployment**, select **Deploy from a branch**.
3. Choose **Branch: main** and **/root**.
4. Click **Save**.
